# Описание

Сервис позволяет генерировать и считывать коды:
- QR
- Code 39
- Code 128
- EAN

При запуске для пользователя создается новая сессия, которая хранит историю запросов.

# Запросы

- `GET` Создать новый код
- `GET` Считать код
- `GET` Получить историю запросов
- `POST` Очистить историю

Недоступные пользователю:  
- `POST` Создать новую сессию
- `POST` Завершить сессию

# Как запускать

Сервер:  
`python3 main.py [--host[=::]] [--port[=8000]]`

Клиент:  
`python3 main.py [--host[=localhost]] [--port[=8000]]`
